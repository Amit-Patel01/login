# 🔐 Login System (PHP & MySQL)

A simple user authentication system developed using **PHP, MySQL, HTML, and CSS**.

This project includes user registration, login authentication, session management, and dashboard access.

## 🎯 Features

- 👤 User Registration  
- 🔑 User Login Authentication  
- 📊 Dashboard Access After Login

---

## ⚙️ How to Run

1. Install XAMPP or WAMP
2. Start Apache and MySQL
3. Import database in phpMyAdmin
4. Update database credentials in `db_config.php`
5. Place project folder inside `htdocs`
6. Run in browser:


---

## 🧠 Working Flow

1. User registers via `Register.html`
2. Data stored in MySQL database
3. Login credentials verified using `login.php`
4. If valid → Redirect to `dashboard.php`
5. Session maintained until logout
6. `logout.php` destroys session
